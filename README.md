# Pickko Shield: Premium Android Privacy & DNS Utility

Pickko Shield is a high-performance, private DNS utility for Android that allows users to switch between top-tier DNS providers while providing real-time ad and tracker protection through a local VPN loopback.

![Dashboard Status](https://via.placeholder.com/400x800.png?text=Pickko+Shield+Dashboard)

## 🚀 Key Features

- **Real-Time DNS Interception**: Catches and filters DNS queries locally on your device.
- **Smart Modes**: Instantly switch between optimized presets:
    - **Balanced**: Standard ad & tracker blocking.
    - **Maximum Privacy**: Zero-logging resolution via Quad9.
    - **Gaming**: Lowest latency routes via Cloudflare.
    - **Family Safe**: Filters adult content and malware.
- **Live Activity Logs**: See which domains your phone is resolving in real-time.
- **Advanced Analytics**: Track your total queries and blocked threats.
- **Network Tools**: Built-in DNS speed tests and leak verification.

## 🛠 Tech Stack

- **Frontend**: React + Tailwind CSS v4
- **Mobile Bridge**: Capacitor
- **Backend**: Native Java (Android VpnService)
- **Monetization**: AdMob Integration (Premium tier removes ads)

## 🔒 Privacy First

Pickko Shield operates as a **local loopback**. Your DNS queries are processed on your device and forwarded directly to your chosen provider. We never log, see, or sell your browsing history.
